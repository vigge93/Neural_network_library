NeuralNetworks

A library by Victor Arvidsson for the programming environment processing. Last update, 02/05/2019.

A library for creating and using neural networks based on Daniel Shiffman's (The coding train) video series on the topic.

Download

Download NeuralNetworks version 1.3.0 in .zip format.

Installation

Unzip and put the extracted NeuralNetworks folder into the libraries folder of your processing sketches. Reference and examples are included in the NeuralNetworks folder.

Keywords matrix, matrices, math, vectors, neural networks, ai, machine learning, neat, evolution

Reference. Have a look at the javadoc reference here. a copy of the reference is included in the .zip as well.

Source. The source code of NeuralNetworks is available at github, and its repository can be browsed here.
